### Local Usage 
#### 1. First install package cross-env globally(Optional)

```
npm i -g cross-env

```
Ensure npm scripts work properly across Linux, Windows, and all environments.

#### 2. install everything else

```
npm install

```

#### 3a. run on localhost

```
npm start

```
Server will be running on http://localhost:3000
